This repository contains file NIR_Converter.html for converting spectral files that have been downloaded from Sagitto's "Sagitto" app (Android) and "Sagitto Lite" app (iOS). 
These spectral files contain absorbance values in JSON format when saved from the app. The NIR Converter provides a very quick and easy way to convert single spectral files from JSON to csv format, either as absorbance or refletance. 
In addition there is the option to add an average column, in the case of batch scans.
